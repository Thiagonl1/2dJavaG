
# 2D Java Game

Juego 2D desarrollado en Java con enfoque en lógica de juego, renderizado y manejo de eventos.

##  Características

- Motor básico de juego en Java
- Renderizado en 2D
- Sistema de input (teclado)
- Loop de juego (game loop)
- Manejo de sprites

## Tecnologías

- Java
- Swing / JavaFX (según implementación)
- Programación orientada a objetos
## Conceptos aplicados

Game loop
Manejo de estados
Colisiones
Renderizado manual

 ├── graphics/
 ├── input/
